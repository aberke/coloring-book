Now that we can count rotations, we can be more precise when we say a square has more symmetry than a rectangle.


We can also see that a square has more rotational symmetry than a triangle, which in turn has more than a rectangle:

A rectangle has only 2 unique rotations, while a triangle has 3, and a square has 4.
